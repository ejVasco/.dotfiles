# .dotfiles
Dot files and packages for my arch (or endeavourOS) install.

The point of this repo is to have a quick, modular setup for whenever I do an install on a new pc.
To be specific, the repo is organized into 6 directories: basic, general, i3wm, hyprland, themes, and scripts. I've also split installing the packages and installing the configs (using symlinks) into separate scripts.
# How to use
1. Clone the repository (personally I clone it in my home directory)
2. Determine which parts of the .dotfiles wanted/needed (table below with more specific details)
    - basic: includes basic mostly terminal utilities
    - i3wm: includes packages specific to my i3wm setup/config
    - hyprland: includes packages specific to y hyprland setup/config
    - general: includes general programs for desktop usage
    - themes: includes themes
